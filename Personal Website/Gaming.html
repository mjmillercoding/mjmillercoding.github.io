A subjective tier list of games I've played!

S Tier

A game that has great concepts that are played out to the fullest potential.

Knights of the Old Republic 1 & 2: This game is great because you can see your actions afecting various parts of your stories.

Escape from Tarkov: I love this game. It is an ultra realistic multiplayer shooter, where everything you use can be lost. The goal of the game is to survive raids in location while getting loot.

The Talos Principle: This game is a must play. If you have ever played the game portal, this game has the same elements that made that game a good puzzle game. On top of that it has a great story that has themes of in-doctrination and morality.

A Tier

Games that have great concepts but are not as captivating due to either lack of key factors, such as story.

The Dark Souls Trilogy: This is a great series to play if you enjoy really tough challenges for the euphoria that comes with completing the challenge.

Portal: This is a classic 3D puzzle game that uses the portal gun tool as a basis for all of the puzzles.

Kenshi: This is a very interessting game. It is a 3rd person RPG that has no real main character. You can pretty much do anything in the game that you want, and any characters you recruit play exactly the same fundementally as your first character.

B Tier

Games that are fairly good, however, I probably wouldn't replay.

Super Mario 64: This is a good game for its time but its not one I would play if not for nostalgia

Trials Evolution and Fusion These are challenging 2.5D games where the slightest mistake can ruin a run.

Call of Duty Black Ops 2: Probably the best COD game in terms of multiplayer.
